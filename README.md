# Projeto API REST com Spring Boot

Este projeto foi desenvolvido como parte do meu portfólio profissional, demonstrando minhas habilidades em desenvolvimento de APIs RESTful com Java e Spring Boot. Aqui estão implementadas boas práticas de desenvolvimento de software, padrões de projeto e arquitetura limpa.

## Tecnologias Utilizadas

- Java 17
- Spring Boot 3.x
- Spring Data JPA
- Hibernate
- Maven
- H2 Database (para desenvolvimento)
- Bean Validation
- Lombok
- ModelMapper

## Funcionalidades

- CRUD completo de lançamentos financeiros
- Filtros de pesquisa com paginação
- Tratamento de erros personalizado
- Validação de dados
- Documentação da API com Swagger/OpenAPI

## Como Executar

1. Clone o repositório
2. Certifique-se de ter o Java 17 e Maven instalados
3. Execute `mvn spring-boot:run`
4. Acesse a documentação da API em: `http://localhost:8080/swagger-ui.html`
